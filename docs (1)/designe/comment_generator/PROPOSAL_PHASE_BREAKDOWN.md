## 3. Phase 1: MVP初期納品（30万円）

### 3.1 目的
> **特定の1顧客分のレポート作成が、このツールで完結できる状態**

### 3.2 スコープ

| 機能 | 詳細 | 開発難易度 |
|------|------|------------|
| **Excelアップロード** | .xlsx/.xlsファイル対応、複数シート切替 | ★★☆☆☆ |
| **表データ表示** | アップロードしたExcelの内容をWeb上で確認 | ★★☆☆☆ |
| **分析範囲選択** | マウスドラッグで分析対象セルを指定 | ★★★☆☆ |
| **月列マッピング** | 前月/当月の列を手動または自動で指定 | ★★★☆☆ |
| **AIコメント生成** | Gemini APIで財務分析コメント生成 | ★★★☆☆ |
| **コメントコピー** | ワンクリックでクリップボードへ | ★☆☆☆☆ |
| **基本UI/UX** | シンプルで使いやすい1画面構成 | ★★☆☆☆ |



### 3.3 含まないもの（Phase 2以降）

- ❌ 顧客管理機能
- ❌ テンプレート保存・再利用
- ❌ 複数ページ一括処理
- ❌ 過去データとの比較
- ❌ パワポ自動出力

### 3.4 納品物

1. Webアプリケーション（デプロイ済み）
2. 簡易マニュアル
3. ソースコード一式

### 3.5 成功基準

| 指標 | 目標 |
|------|------|
| 1ページのコメント生成 | 30秒以内 |
| 生成コメント品質 | 軽微な編集で使用可能 |
| 対象顧客レポート | 全ページ作成可能 |

### 4.2 カテゴリA: 運用効率化機能

> **毎月の作業を楽にする**ための機能群

| 機能 | 効果 | 難易度 |
|------|------|--------|
| **テンプレート保存** | 顧客ごとの設定を記憶、次月は設定不要 | ★★★☆☆ |
| **前回設定の復元** | 前月の範囲選択・マッピングを自動適用 | ★★★☆☆ |
| **複数ページ一括処理** | 20ページ分を一度にアップロード→生成 | ★★★★☆ |
| **コメント履歴管理** | 過去の生成コメントを参照可能 | ★★☆☆☆ |
| **プロンプトカスタマイズ** | 顧客ごとのトーン・表現を調整 | ★★☆☆☆ |
| **キーボードショートカット** | 操作の高速化 | ★☆☆☆☆ |

**推奨実装順**: テンプレート保存 → 前回設定復元 → 複数ページ一括

---

### 4.3 カテゴリB: 対象顧客拡大対応

> **複数顧客に対応**するための機能群

| 機能 | 効果 | 難易度 |
|------|------|--------|
| **顧客管理画面** | 顧客情報の登録・一覧表示 | ★★☆☆☆ |
| **顧客別テンプレート** | 顧客ごとに異なるフォーマット対応 | ★★★☆☆ |
| **顧客切替UI** | ワンクリックで顧客を切替 | ★★☆☆☆ |
| **フォーマット自動検出** | Excel構造から顧客を推定 | ★★★★☆ |
| **顧客別レポート履歴** | 顧客ごとの過去レポート閲覧 | ★★☆☆☆ |

**推奨実装順**: 顧客管理画面 → 顧客別テンプレート → 切替UI

---

### 4.4 カテゴリC: 高度化・品質向上

> **AI精度向上・分析機能の拡張**

| 機能 | 効果 | 難易度 |
|------|------|--------|
| **前月比自動ハイライト** | 大きな変動を視覚的に強調 | ★★★☆☆ |
| **コメント品質フィードバック** | 修正内容をAIに学習させる | ★★★★☆ |
| **複数期間比較** | 前月だけでなく前年同月等との比較 | ★★★☆☆ |
| **グラフ・チャート生成** | 数値からグラフを自動生成 | ★★★☆☆ |
| **異常値検出** | 通常と異なる数値を自動検出 | ★★★★☆ |
| **要因分析** | 変動の要因を推定・言語化 | ★★★★★ |

**推奨実装順**: 前月比ハイライト → 複数期間比較 → 異常値検出
