## クイックサマリー

### 目的
Excelの財務数値表から、前月比較を踏まえた財務分析コメントをAIで自動生成し、パワーポイントへの納品資料作成を効率化する。

### 開発フェーズ

| Phase | 期間 | 内容 |
|-------|------|------|
| **Phase 1 (MVP)** | 2-3週間 | Excel入力→範囲選択→コメント生成→コピー |
| **Phase 2** | 2-3週間 | 顧客管理、テンプレート保存・再利用 |
| **Phase 3** | 2-3週間 | 高度機能（項目サジェスト、管理画面） |
| **Phase 4** | 将来 | パワポ自動生成 |

### 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | Vite + React + TypeScript |
| Excelパース | xlsx ライブラリ（クライアントサイド） |
| バックエンド | Node.js + Express (Cloud Run) |
| AI | Google Gemini API |
| データストア | Firestore (Phase 2以降) |
