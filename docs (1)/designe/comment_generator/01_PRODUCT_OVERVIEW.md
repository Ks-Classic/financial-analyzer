# 財務コメント自動生成機能 - 企画書

## 1. エグゼクティブサマリー

### 1.1 プロダクト名
**財務コメント自動生成機能**（Financial Comment Generator）

### 1.2 一言で言うと
> Excelの財務数値表をアップロードするだけで、前月比較を踏まえた財務分析コメントをAIが自動生成し、パワーポイントへの貼り付けを効率化するツール

---

## 2. 解決する課題

### 2.1 現状の業務フロー（As-Is）

```
1. Excelで財務データを集計
2. 前月との比較を手作業で分析
3. 分析結果を文章化（コメント作成）← ★ここが時間かかる
4. ページごとにフォーマットに合わせてコメント調整
5. パワポに表とコメントをコピペ
6. 20ページ分繰り返し
7. 複数顧客分繰り返し
```

### 2.2 ペインポイント

| 課題 | 影響 |
|------|------|
| **コメント作成に時間がかかる** | 1顧客あたり数時間、月次で膨大な工数 |
| **顧客ごとにフォーマットが異なる** | 都度調整が必要、ミスが発生しやすい |
| **前月比較の洞察抽出が属人的** | 担当者によって品質にばらつき |
| **コピペ作業が煩雑** | 単純作業だが時間を消費 |

### 2.3 このプロダクトで解決すること

| Before | After |
|--------|-------|
| 手作業で前月比較・分析 | AIが自動で変動を検出・分析 |
| 1ページ10-15分のコメント作成 | ワンクリックで生成、編集のみ |
| 顧客ごとにゼロから作業 | テンプレート再利用で効率化 |
| コピペ繰り返し | ボタン一つでクリップボードへ |

---

## 3. ターゲットユーザー

### 3.1 プライマリユーザー

**財務レポート作成担当者**

- 月次で顧客向け財務レポートを作成
- 複数顧客（10社以上）を担当
- 顧客ごとに異なるフォーマットのレポートを作成
- パワーポイントでの納品資料作成

### 3.2 ユーザーの特徴

| 属性 | 詳細 |
|------|------|
| スキル | Excel操作に習熟、財務知識あり |
| 業務頻度 | 月1回（月初に集中） |
| 時間的制約 | 納期がタイト、効率化ニーズ高 |
| 品質要求 | 高い（顧客納品物のため） |

---

## 4. 提供価値

### 4.1 定量的価値

| 指標 | Before | After | 削減率 |
|------|--------|-------|--------|
| 1顧客のコメント作成時間 | 3-4時間 | 30分-1時間 | **75%削減** |
| 月間作業時間（10顧客） | 30-40時間 | 5-10時間 | **75%削減** |
| ミス発生率 | 5-10% | 1-2% | **80%削減** |

### 4.2 定性的価値

- **品質の均一化**: AI生成により担当者間のばらつき解消
- **洞察の深化**: 人間が見落としがちな変動をAIがピックアップ
- **ストレス軽減**: 単純作業からの解放
- **スケーラビリティ**: 顧客数増加に対応しやすい

---

## 5. 競合・代替手段

### 5.1 現在の代替手段

| 手段 | メリット | デメリット |
|------|----------|------------|
| 手作業 | カスタマイズ自由 | 時間がかかる、属人的 |
| Excel VBA | 自動化可能 | メンテ困難、AI分析不可 |
| ChatGPT直接利用 | AI活用可能 | 毎回プロンプト入力、フォーマット不統一 |

### 5.2 本プロダクトの優位性

- **業務フローに統合**: アップロード→生成→コピーのシームレスな体験
- **テンプレート管理**: 顧客ごとの設定を保存・再利用
- **Excel直接読み込み**: データ入力不要、そのまま分析
- **フォーマット遵守**: 顧客要件に沿った出力

## 6. スコープ

### 6.1 MVP（Phase 1）でやること

| 機能 | 説明 |
|------|------|
| Excelアップロード | 複数シート対応 |
| 範囲選択 | 分析対象の表を指定 |
| 月列自動検出 | 前月/当月を自動判定 |
| AIコメント生成 | Gemini APIで分析・生成 |
| コピー機能 | クリップボードへコピー |

### 6.2 MVPでやらないこと（Phase 2以降）

| 機能 | 理由 |
|------|------|
| 顧客管理 | まずは動くものを優先 |
| テンプレート保存 | MVP後に追加 |
| パワポ自動生成 | 技術的ハードル高、将来対応 |
| 画像（表）の読み取り | Excel優先 |

## 8. リスクと対策

| リスク | 影響度 | 対策 |
|--------|--------|------|
| AI生成品質が低い | 高 | プロンプト設計の徹底、人間編集を前提とした設計 |
| Excelフォーマットの多様性 | 中 | 範囲選択UI、顧客別テンプレート |
| 月ずれ対応が複雑 | 中 | ヘッダー自動検出、動的マッピング |
| ユーザー操作が複雑になる | 中 | UXテスト、シンプルなフロー設計 |
