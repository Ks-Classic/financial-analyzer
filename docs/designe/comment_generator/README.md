## クイックスタート

### 環境設定

```bash
# 環境変数設定
cp .env.example .env.local
# GEMINI_API_KEY=your-api-key を設定

# 開発サーバー起動
pnpm dev
```

### 使い方

1. PDFをアップロード
2. ページを選択
3. 各ページに今月データ画像を貼り付け
4. プロンプトを確認・調整
5. 「一括生成」をクリック
6. 生成結果を確認・編集・エクスポート

## 主要機能

| 機能 | 状態 | 説明 |
|------|------|------|
| PDFコメント抽出 | ✅ 実装済 | PDFから前月コメントを自動抽出 |
| 画像ペースト入力 | ✅ 実装済 | Ctrl+V またはドラッグ&ドロップ |
| 一括キャッシュ生成 | ✅ 実装済 | 全ページを一括キャッシュ→並列生成で高速 |
| チャット修正 | ✅ 実装済 | ページごとに自然言語で修正指示 |
| 顧客別設定保存 | ⚠️ 部分実装 | localStorage保存（ブラウザ限定） |
| 複数人共有 | 🔜 将来対応 | Supabase連携で実現予定 |
